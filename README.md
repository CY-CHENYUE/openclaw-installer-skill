# OpenClaw Installer Skill

一个 Claude Agent Skill，一键安装部署 OpenClaw（原 Clawdbot/Moltbot）私人 AI 助手。

## 使用方法

克隆到 Claude Code skills 目录，然后说：**帮我安装 OpenClaw**

## 功能

- 16 个 AI 提供商（含 DeepSeek/Kimi/智谱 GLM 等国内模型）
- 7 个消息通道（Telegram/Discord/微信/飞书等）
- 所有命令经实际安装测试验证
